# 🤖 بوت الملصقات الاحترافي - Telegram Sticker Bot

بوت تيليجرام احترافي لإنشاء وإدارة الملصقات مع دعم كامل للغة العربية والإنجليزية.

## ✨ المميزات

- 🖼 **تحويل الصور لملصقات**: تحويل الصور العادية إلى ملصقات تيليجرام مع دعم الخلفيات الشفافة
- 🎥 **تحويل الفيديو لملصقات متحركة**: تحويل مقاطع الفيديو القصيرة إلى ملصقات متحركة
- 📦 **إدارة حزم الملصقات**: إنشاء وإدارة ومشاركة حزم الملصقات
- 🌐 **دعم متعدد اللغات**: العربية والإنجليزية
- 📊 **إحصائيات مفصلة**: تتبع استخدامك للملصقات والحزم
- 🔒 **نظام صلاحيات**: دعم كامل لإدارة المستخدمين والصلاحيات
- 🚀 **نشر سهل**: جاهز للنشر على Railway ومنصات أخرى

## 📋 المتطلبات

- Python 3.12+
- FFmpeg (للملصقات المتحركة)
- Token بوت تيليجرام من [@BotFather](https://t.me/BotFather)
